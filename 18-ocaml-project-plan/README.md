# ocaml-project-plan

OCaml CLI template that builds a project plan with milestones.

This repo is intentionally written in **OCaml** (not Python/JS) so GitHub shows a different language mix.

## Run
```bash
ocamlc main.ml -o app
./app
```

## Input format
Put your notes in `input.txt` (or pipe stdin). Output is printed to stdout.
